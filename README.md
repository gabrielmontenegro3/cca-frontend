# CCA - Sistema

Sistema web desenvolvido com React, TypeScript, Tailwind CSS e Axios.

## Tecnologias

- React 18
- TypeScript
- Tailwind CSS
- Axios
- Vite

## Instalação

```bash
npm install
```

## Executar em desenvolvimento

```bash
npm run dev
```

## Build para produção

```bash
npm run build
```

## Estrutura do Projeto

- `src/components/` - Componentes reutilizáveis (Navbar, Sidebar)
- `src/pages/` - Páginas do sistema
- `src/App.tsx` - Componente principal
- `src/main.tsx` - Ponto de entrada da aplicação

## Funcionalidades

- **Visão Geral** - Painel principal com resumo de informações
- **Empreendimento** - Informações sobre o empreendimento
- **Meu Imóvel** - Detalhes do imóvel do usuário
- **Garantias** - Gerenciamento de garantias de produtos
- **Preventivos** - Agendamento e acompanhamento de manutenções preventivas
- **Manutenção e Uso** - Orientações de manutenção e uso adequado
- **Locais** - Informações sobre locais importantes do condomínio
- **Produtos** - Lista de produtos instalados no imóvel
- **Fornecedores** - Informações dos fornecedores
- **Contatos** - Lista de contatos importantes
- **Documentos** - Acesso e download de documentos
- **Perguntas Frequentes** - FAQ com filtro por categoria
- **Sobre nós** - Informações sobre o sistema
- **Boletim Informativo** - Notícias e atualizações
- **Assistência Técnica** - Solicitação e acompanhamento de assistência técnica

## Características

- Sidebar com navegação entre todas as funcionalidades
- Navbar fixa no topo
- Interface responsiva com Tailwind CSS
- Integração com Axios para chamadas de API
- TypeScript para type safety
